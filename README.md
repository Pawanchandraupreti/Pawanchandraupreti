## 🚀 Featured Projects

### 🔍 Network Intrusion Detection System

> Built a real-time IDS that monitors live network traffic and flags threats **before damage occurs.**  
> Combines **rule-based signatures** with a trained **ML model** to catch both known attacks and zero-day anomalies.

| Capability | Detail |
|------------|--------|
| 📡 Live Packet Capture | Real-time traffic monitoring |
| 🧠 ML Detection | Anomaly-based threat classification |
| 📋 Rule Engine | Signature matching for known attack patterns |
| 🚨 Alert System | Instant logging of suspicious activity |

**Impact:** Dual-layer detection drastically reduces false negatives — catching threats that rule-only systems miss.

`Python` `Scikit-Learn` `Scapy` `Logging` `Machine Learning`

### 🔓 Brute Force Login Tool

> Engineered a controlled authentication stress-testing tool targeting demo environments.  
> Executed **500+ simulated login attempts** across multiple endpoints — uncovering critical weaknesses in rate-limiting and lockout mechanisms.

| Metric | Result |
|--------|--------|
| 🔴 Failed Attempts | 72% |
| 🟢 Successful Bypasses | 18% |
| 🟡 Blocked by System | 10% |

**Key Finding:** 1 in 5 attempts bypassed authentication — exposing the target's lack of proper account lockout policy.

`Python` `Requests` `JSON` `HTTP Protocols` `Automation`
